# 🎬 IMDB Movie Data Analysis – SQL Project

## 📌 Project Overview

This project analyzes **IMDB movie datasets using SQL** to generate insights that can help a production company (**RSVP Movies**) make data-driven decisions for their upcoming movie projects.

The analysis explores **movie trends, genres, production houses, directors, actors, and ratings** to identify successful patterns in the film industry.

The goal is to answer key business questions such as:

* Which movie genres perform best?
* Which production houses produce the most successful movies?
* Which directors and actors consistently appear in highly rated films?
* Which trends can guide RSVP Movies in selecting their next project?

---

# 📊 Data Source

The dataset is derived from **IMDB movie data**, consisting of multiple relational tables.

### Tables Used

* `movie` – Contains movie information such as title, year, duration, language, and production company
* `genre` – Movie genre classifications
* `ratings` – Movie ratings and vote counts
* `names` – Information about actors and directors
* `director_mapping` – Maps movies to directors
* `role_mapping` – Maps actors to movies

These tables were analyzed using **SQL queries to extract insights about the movie industry**.

# 🔍 Analysis Performed

## 1️⃣ Data Exploration

Initial analysis included:

* Checking total rows in each table
* Identifying columns with **null values**
* Understanding the schema relationships

Example tasks:

* Row count analysis
* Missing data detection
* Table structure exploration

---

## 2️⃣ Movie Release Trends

Analyzed how movies were released over time:

* Total movies released per **year**
* Monthly release trends
* Growth of movie production over time

This helped identify **industry production patterns**.

---

## 3️⃣ Genre Analysis

Examined which movie genres dominate the industry.

Key analysis:

* Number of movies by genre
* Average ratings per genre
* Top genres with **highly rated movies**

Insight:
The **Drama genre had the highest number of movie releases**, indicating strong audience demand.

---

## 4️⃣ Ratings & Performance Analysis

Evaluated movie performance using:

* Average movie ratings
* Median ratings
* Total vote counts

This helped identify **critically successful films and production companies**.

---

## 5️⃣ Director Analysis

Identified directors who consistently produce **highly rated movies**.

Example insight:

* **James Mangold** emerged as a top director among highly rated films in the most popular genres.

---

## 6️⃣ Actor & Actress Analysis

Analyzed actors whose movies consistently receive **high ratings**.

Key findings:

Top actors based on ratings:

* **Mammootty**
* **Mohanlal**

Top Indian actors and actresses with strong ratings:

* **Vijay Sethupathi**
* **Fahad Faasil**
* **Taapsee Pannu**
* **Divya Dutta**

---

## 7️⃣ Production House Analysis

Identified production houses producing **successful movies**.

Top production houses:

* Dream Warrior Pictures
* National Theatre Live
* Marvel Studios

These companies have produced several **highly rated and popular films**.

---

# 📈 Key Business Insights

Based on the analysis, the following strategic recommendations were made for **RSVP Movies**:

* Focus on producing **Drama genre films**, as they dominate the industry.
* Collaborate with successful production houses like **Dream Warrior Pictures**.
* Consider hiring experienced directors like **James Mangold**.
* Work with highly rated actors such as **Mammootty** and **Mohanlal**.
* Explore partnerships with globally successful studios like **Marvel Studios**.

These insights help support **data-driven decision-making in movie production**.
